## ðŸŽ¨ Design Philosophy: Glassmorphism
This project utilizes **Glassmorphism** (frosted glass aesthetic) to create a modern, depth-based UI.
* **Pure CSS:** No external CSS frameworks (Bootstrap/Tailwind) were used, ensuring maximum performance and lightweight load times.
* **Visual Hierarchy:** Uses translucency and blur (`backdrop-filter`) to establish hierarchy between the background and content cards.
* **Responsiveness:** Fully responsive grid layout that adapts seamlessly from desktop to mobile.

## ðŸš€ Deployment Instructions

### Option 1: GitHub Pages (Recommended)
1.  Push `index.html` to the `main` branch.
2.  Go to **Settings** > **Pages**.
3.  Select `main` branch and click **Save**.
4.  Your site will be live at `https://<your-username>.github.io/<repo-name>`.

### Option 2: AWS S3 (Static Website Hosting)
1.  Create a public S3 bucket.
2.  Enable **Static Website Hosting** in Properties.
3.  Upload `index.html`.
4.  Apply a Bucket Policy for public read access.
